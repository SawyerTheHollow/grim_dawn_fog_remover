## En
Utility for quickening the process of unlocking fog on new characters throuh copying it from existing characters. Select the character from which fog will be copied in the right dropdown and the character which fog file will be overwritten in the left dropdown, then press the button below. There is no english localization, but if someone might need it, file an issue, i will add it.
## Ru
Утилита для ускорения процесса раскрытия тумана карты для новых персонажей путём копирования тумана с существующих персонажей. В правом списке нужно выбрать персонажа, с которого будет скопирован файл тумана, а в левом списке - персонажа, файл тумана которого будет заменён. Далее нужно просто нажать кнопку ниже.
